## Gerar db Prisma
```
yarn prisma migrate dev
```

## Instalar dependências
```
npm install
```

## Iniciar o front-end
```
npm start
```

## Iniciar o back-end
```
yarn dev
```

## Arquivo .env do back-end
```
DATABASE_URL="mysql://root:1234@localhost:3306/imobiliaria"
```
